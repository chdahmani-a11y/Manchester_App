# Application illustrative du code Manchester

## Description
Cette application web permet de visualiser le **code Manchester** utilisé dans les réseaux informatiques.
L'utilisateur entre un code binaire et l'application affiche :
- La conversion en code Manchester (texte)
- Le graphique du signal correspondant

## Principe du code Manchester
| Bit binaire | Code Manchester | Transition |
|-------------|----------------|------------|
| 1 | 01 | Bas → Haut |
| 0 | 10 | Haut → Bas |

## Comment utiliser l'application ?
1. Entrez un code binaire (exemple : `1010`)
2. Cliquez sur "Générer le signal"
3. Le code Manchester et le graphique s'affichent automatiquement

## Technologies utilisées
- HTML5
- CSS3
- JavaScript (Canvas API)
